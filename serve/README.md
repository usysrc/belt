# serve

A simple file server.

## Usage

```
serve [flags]
```

### Flags

| Flag | Description | Default |
|---|---|---|
| `-p`, `--port` | Port for the file server | `8080` |

## Installation

### From source

```
go install github.com/user/belt/serve@latest
```
